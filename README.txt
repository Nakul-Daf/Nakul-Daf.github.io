# Nakul Daf | Data Engineer Portfolio 🚀

Welcome to the source code repository for my personal portfolio website. This site serves as an interactive digital resume and a showcase of my enterprise data engineering projects, architectures, and methodologies.

🌐 **Live Website:** [nakul-daf.github.io](https://nakul-daf.github.io/)

## 📖 Overview
As a Data Engineer, I specialize in building scalable, reliable cloud data platforms using Azure Databricks, PySpark, Snowflake, and Delta Lake. This portfolio is designed to provide a high-level overview of my 4+ years of professional experience, core technical skills, and featured case studies.

## 🛠️ Portfolio Tech Stack
While my day-to-day work involves big data processing and cloud architecture, I built this website to be fast, responsive, and highly interactive using modern front-end tools:
* **HTML5:** Semantic and accessible structure.
* **Tailwind CSS:** Utility-first CSS framework for rapid, responsive styling.
* **AOS (Animate On Scroll):** Smooth scroll-triggered reveal animations.
* **Vanilla-Tilt.js:** Premium 3D tilt interactions on interactive elements.
* **Typed.js:** Dynamic typewriter text effects.

## 📁 Repository Structure
* `index.html`: The main, single-page portfolio layout.
* `/images/`: Contains optimized image assets, including my profile picture and project thumbnails.
* `README.md`: Project documentation (this file).
